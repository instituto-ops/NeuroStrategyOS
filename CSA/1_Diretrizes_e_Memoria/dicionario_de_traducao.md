# 📘 Dicionário de Tradução Dinâmico (Psicologia -> TI)

A Metodologia Antigravity traduz a intenção clínica e psicológica do Arquiteto diretamente para a execução arquitetural de software.

| Conceito Clínico / Sistêmico (O Prompt) | Materialização Arquitetural Executada pela IA |
| :---- | :---- |
| **"Individuação"** (*Jung*) | Desacoplamento de microsserviços. Banco de dados próprio por domínio. |
| **"Memória Implícita vs. Explícita"** | Cache (Redis) vs. Banco Vetorial/RAG. |
| **"Reflexos Instintivos"** | Event Listeners assíncronos via WebSockets (arquitetura reativa). |
| **"Abrandamento Crítico"** | Circuit Breakers, Rate Limiting, Auto-scaling. |
| **"Espelhamento"** (*Mirroring*) | Digital Twins, Telemetria e Observabilidade em tempo real. |
| **"Poda Sináptica"** | Refatoração autônoma: remoção de dead code e dependências extras. |
| **"Contenção"** (*Containment*) | Fail-safe, sandboxing, graceful degradation. |
| **"Sincronicidade"** | Orquestração de eventos distribuídos e estados compartilhados. |
| **"Projeção"** | Virtualização de sistemas de terceiros ou mocks inteligentes. |
