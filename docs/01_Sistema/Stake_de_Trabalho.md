# 🛠️ NeuroEngine OS: Stake de Trabalho (Stack v2.0)

Esta é a definição técnica e metodológica da infraestrutura que sustenta o **NeuroEngine OS**. O sistema foi projetado para alta performance em **Neuromarketing de Precisão** e **Governança Clínica do Dr. Victor Lawrence**.

---

## 🏛️ 1. Core Tecnológico (The Tech Stack)

### 🎨 Frontend: UI/UX de Elite
*   **Linguagem:** Vanilla HTML5 & JavaScript (ES6+).
*   **Design System:** CSS Moderno com suporte a **Glassmorphism**, **Dark Mode Nativo** e variáveis dinâmicas de estilo.
*   **Iconografia:** [Lucide Icons](https://lucide.dev/) (Injetados via JS).
*   **Captura de Layout:** [html2canvas](https://html2canvas.hertzen.com/) para espelhamento fiel de previews para o WordPress.
*   **Arquitetura de Estado:** Gestor de orquestração global (`app.js`) com persistência em LocalStorage.

### ⚙️ Backend: O Cérebro do Sistema
*   **Runtime:** Node.js (Ambiente Windows VM).
*   **Framework:** Express.js (API REST & Servidor de Arquivos).
*   **Comunicação Real-time:** WebSockets (`ws`) para logs de auditoria e monitoramento de latência ("O Pulso").
*   **Ingestão de Mídia:** Cloudinary (CDN) & Filesystem Local sincronizado.
*   **Processamento de Documentos:** `mammoth` (DOCX), `pdf-parse` (PDF) e `multer` (Multipart Uploads).

## 🧠 2. Inteligência Artificial: Governança Gemini 2.5

O NeuroEngine OS utiliza a família **Gemini 2.5** como motor de orquestração única, sob a **Norma de Economia de Tokens**.

| Nível | Motor | Função Principal |
| :--- | :--- | :--- |
| **Poder Crítico** | `gemini-2.5-pro` | Auditoria Clínica (CRP), Reflexão de Elite, Análise de Longo Contexto (2M+ tokens). |
| **Operacional** | `gemini-2.5-flash` | **Default (Abidos Standard)**. Geração de Copy, Hubs & Spokes, SEO Mapping (1M tokens). |
| **Interativo** | `gemini-2.5-flash-lite` | Chat de suporte rápido, summarização de logs e triagem de metadados. |

## 🏛️ 3. Metodologia: O Protocolo Abidos v5.6

A "Stake" não é apenas código; é o **Manejo Clínico de Marketing**:

*   **Hub & Spoke Architecture:** Dominação de autoridade regional vs. nacional.
*   **Copy Fatiado:** Abandono do H1 único por blocos contextuais (Kicker, H1, Subtitle).
*   **Neuromarketing Ético:** Filtro anti-gatilhos de "milagre" ou "cura", alinhado ao CFP (Conselho Federal de Psicologia).
*   **Data-Driven SEO:** Cruzamento de ferramentas externas (Search Console/GA4) com o Banco de Dados de Keywords do OS.

## 🛡️ 4. Infraestrutura de Governança

1.  **Antigravity Watchdog:** Monitoramento contínuo de integridade e deploy automático para Vercel/Next.js.
2.  **Silo Management:** Organização de conteúdo em categorias estritas (Teoria do Apego, Neurodivergência, etc.) para evitar canibalização.
3.  **Auditoria 360°:** O sistema não publica sem o "Selo Verde" (Green Status) de Neuromarketing e Ética Clínica.
