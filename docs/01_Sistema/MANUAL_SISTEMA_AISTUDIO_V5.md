### 3. DICIONÁRIO DE INJEÇÃO (PADRÃO ABIDOS)
Toda saída deve ser um JSON plano preenchendo estas chaves obrigatórias:

#### A. Camada SEO Invisível (Camada Oculta para Robôs)
* `{{seo_h1_tecnico}}`: Palavra-Chave Alvo + Localização + Nome do Profissional.
* `{{seo_resumo_indexacao}}`: Resumo semântico denso (máx. 160 caracteres).
* `{{secaoX_h2}}`: Títulos técnicos para hierarquia de rastreamento.

#### B. Camada Visual (UI Artística para Humanos)
* `{{ui_titulo_hero}}`: Título poético, curto e de alto impacto emocional.
* `{{hero_subtitulo}}`: Copy de rapport imediato focada no espelhamento da dor.
* `{{artigo_paragrafo_introdutorio}}`: Lead focado em Pacing (acompanhamento fenomenológico).
* `{{citacao_destaque}}`: Frase de impacto em itálico para quebra de padrão visual.

#### C. Variáveis de Artigo (Estrutura de Conteúdo)
* `{{artigo_h1_titulo}}`: Título visual do artigo (injetado em div/span).
* `{{artigo_subtitulo}}`: Subtítulo de apoio.
* `{{artigo_imagem_url}}`: URL da imagem de destaque.
* `{{secaoX_p1}}`, `{{secaoX_p2}}`: Parágrafos de conteúdo.
* `{{area_dinamica_extra}}`: Slot para blocos modulares.
